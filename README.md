# Análise de Vendas

**Objetivo:**

Este projeto tem como objetivo principal realizar uma análise a respeito das vendas do E-Comerci Meteora. Meteora é um E-comerci especializado em roupas e acessórios.

**Propósito:**
A análise consiste em criar um relatório Financeiro e vendas à equipe de gerência para embasar a tomada de decisões estratégicas, visualizar os desempenho das vendas, buscar de melhorias. Através da análise de dados e da criação de um dashboard estratégico em Finanças, o projeto busca:
