# Análise de Vendas

**Objetivo:**

Este projeto tem como objetivo principal realizar uma análise a respeito das vendas do E-Comerci Meteora. Meteora é um E-comerci especializado em roupas e acessórios.
O presente projeto tem como objetivo principal realizar uma análise abrangente das vendas do E-Commerce Meteora, loja online especializada em roupas e acessórios. Através de um estudo aprofundado dos dados, busca-se fornecer à equipe de gerência informações valiosas para embasar decisões estratégicas, otimizar o desempenho das vendas e identificar oportunidades de crescimento.

**Propósito:**
A análise consiste em criar um relatório Financeiro e vendas à equipe de gerência para embasar a tomada de decisões estratégicas, visualizar os desempenho das vendas, buscar de melhorias. Através da análise de dados e da criação de um dashboard estratégico em Finanças, o projeto busca:

**Mensurar o desempenho das vendas:** Através da criação de indicadores-chave de performance (KPIs), foi possível avaliar o desempenho das vendas em diferentes aspectos, como: Receitas, despesas, quantidade vendida, sazonalidades, rentabilidade por categorias e produtos.

**Identificação de tendências e padrões:** A análise dos dados permitirá identificar tendências sazonais, de comportamento do consumidor e de mercado, fornecendo insights valiosos para a criação de estratégias direcionadas.

**Tomada de decisões assertivas:** O relatório fornecerá à equipe de gerência as informações necessárias para tomar decisões estratégicas com base em dados concretos, aumentando as chances de sucesso.
